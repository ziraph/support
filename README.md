# Ziraph support

Help, questions, and feedback for [Ziraph](https://ziraph.com) - honest local AI profiling for Apple Silicon.

Pick the channel that fits what you need:

## 💬 [Discussions](https://github.com/ziraph/support/discussions) - questions and conversation

- **[Q&A](https://github.com/ziraph/support/discussions/categories/q-a)** - stuck, or not sure whether it is a bug? Ask here.
- **[General](https://github.com/ziraph/support/discussions/categories/general)** - share a run, compare results, or talk about local AI on Apple Silicon.
- **[Announcements](https://github.com/ziraph/support/discussions/categories/announcements)** - releases and news.

## 🐛 [Issues](https://github.com/ziraph/support/issues) - bugs and feature requests

- **Found a bug?** [Open one](https://github.com/ziraph/support/issues/new/choose) - the template asks for the chip, the command, and the output needed to reproduce it.
- **Want a feature?** Open an issue and describe what you are trying to measure.

Not sure which? A Discussion is a safe start; it can move to an Issue if it turns out to be a bug.

---

**Product, full docs, and the blog** are at **[ziraph.com](https://ziraph.com)**. Ziraph is in private beta; you can request access from the site.
